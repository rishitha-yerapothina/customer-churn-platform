
Customer Churn Prediction Platform

Overview:
Customer Churn Prediction Platform is a full-stack machine learning application that predicts whether a customer is likely to leave a telecom service.

The system uses a Logistic Regression model trained on the Telco Customer Churn dataset and provides:
1.Churn prediction
2.Churn probability score
3.Risk classification
4.Key risk factors
5.Retention recommendations
6.Analytics dashboard

Features:
Customer Churn Prediction

Predict whether a customer is likely to churn using a trained machine learning model.

Risk Classification

Customers are categorized as:

Low Risk
Medium Risk
High Risk
Churn Probability

Displays probability score generated by the model.

Key Risk Factors

Highlights factors associated with churn:

Contract type
Customer tenure
Payment method
Retention Recommendations

Provides recommended actions to improve customer retention.

Analytics Dashboard

Includes:

Model Accuracy
Total Customers
Churn Cases
Retention Rate
Churn Distribution
Contract Risk Analysis
